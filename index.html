Surgimento dos computadores

(Voltar)

Um dos primeiros computadores foi o Harvard Mark I, foi utilizado e produzido pelos Estados Unidos no ano de 1944, mas não era um computador muito potente, era utilizado para cauculos e demorava cerca 3 a 5 segundos para resolver a multiplicação. Um dos outros percursores foi o Colossus fabricado em 1946 por britanicos para ser utilizadp para criptoanálise de cifra. Outro computador criado também em 1946 foi o ENIAC, sua tecnologia mais avançada trouxe um diplay digital de grande escala, e uma de suas funções era cáuculos balísticos. Assim iniciou-se o processo de formação de computadores, e seu tamanho foi ficando cada vez menor em comparação aos anteriores, pois os avanços tecnologicos foneceram peças menores _ resistores, micro-processadores... etc,etc,etc_ e assim chegou ate onde estamos hoje.

Tags e suas funções

(Voltar)

Para iniciarmos em HTML precissamos aprender e também saber como é que cada tag fuciona e suas respectiva função.

Exemplos:

  1. input -> está tag pode ser utilizada para adicionar campo de texto, mais utilizadas para formularios.

  2. blockquote -> Tag muito utilizada quando queremos fazer citações de algum livro ou frases importantes de pessoas.

  3. mark -> utilizada como marcador de texto, para marca frases ou palvras que sejão importantes.

  4. small -> Tem a função de criar uma espécie de butão para gerar link para ser utilizado como âcoragem.

HTML é uma linguagem?

(Voltar)

Quando estamos iniciando em programação começamos pelo basico,HTML, CSS, entre outros, mas HTML é uma linguagem? Não! O HTML não pode ser uma linguagem, pois o mesmo não possui funcionalidades dinâmicas como às verdadeiras linguagens, Javascript, Java, PhP, etc.
